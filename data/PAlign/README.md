# PAlign Dataset
Эти датасеты взяты из репозитория [PAlign](https://github.com/zhu-minjun/PAlign).

## Download Test-Set Dataset
Это датасеты для воспроизведения работы, включая части IPIP-NEO и Dark-Triad из статьи

### IPIP-NEO
- **IPIP-NEO-ItemKey.xls**: Содержит ключи элементов для инвентаря личности IPIP-NEO.
- **mpi_120.csv**: Ответы на анкету IPIP-NEO-120.
- **mpi_300.csv**: Ответы на анкету IPIP-NEO-300.
- **mpi_300_split.json**: Test-Set разбиение для датасета PAPI (IPIP-NEO).
- **Test-set.json**: Test-Set данные для датасета PAPI (IPIP-NEO).

### Dark-Triad
- **dark_triad-ItemKey.xls**: Содержит ключи элементов для инвентаря личности Dark-Triad.
- **dark_triad_Test_set.json**: Test-Set данные для датасета PAPI (Dark-Triad).
- **dark_triad_split.json**: Test-Set разбиение для датасета PAPI (Dark-Triad).
